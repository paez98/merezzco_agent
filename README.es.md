# 🤖 Agente de Ventas con IA – Cotización y Gestión de Pedidos

[Docs. en ingles 🇺🇸](README.md)

Agente conversacional diseñado para automatizar la atención al cliente, generar cotizaciones personalizadas, ofrecer recomendaciones inteligentes y gestionar pedidos dentro de un flujo comercial real.

Este proyecto demuestra cómo construir un sistema backend basado en LLM utilizando principios de arquitectura limpia y procesamiento estructurado del contexto conversacional.

---

## 🎯 Propósito del Proyecto

No es un simple chatbot.

Es un sistema de IA orientado a negocio capaz de:

- Procesar múltiples mensajes del usuario
- Mantener el contexto cronológico
- Detectar intención
- Generar respuestas comerciales estructuradas
- Gestionar flujos de pedidos
- Integrarse en pipelines automatizados

Simula un asistente de ventas real operando a través de plataformas de mensajería.

---

## 🚀 Capacidades Principales

### 1️⃣ Manejo de Contexto Conversacional
- Recibe múltiples mensajes
- Los ordena cronológicamente
- Construye historial estructurado
- Preserva coherencia semántica

### 2️⃣ Generación de Cotizaciones Personalizadas
- Extrae cantidades y restricciones
- Interpreta requerimientos implícitos
- Genera cotizaciones dinámicas
- Sugiere configuraciones adecuadas

### 3️⃣ Recomendaciones Inteligentes
- Detecta información faltante
- Formula preguntas estratégicas
- Sugiere opciones óptimas
- Adapta respuestas según el flujo

### 4️⃣ Gestión de Pedidos
- Confirma datos del cliente
- Resume detalles del pedido
- Prepara salida estructurada para sistemas externos
- Listo para integración con pagos

---

## 🧠 Arquitectura del Sistema

Basado en principios de Arquitectura Limpia:

- Separación de responsabilidades
- Capa de servicios
- Abstracción de repositorios
- Lógica de dominio aislada
- Infraestructura desacoplada

Flujo conversacional:

1. Recepción de mensajes
2. Ordenamiento cronológico
3. Construcción del historial
4. Análisis de intención
5. Aplicación de reglas de negocio
6. Generación de respuesta estructurada
